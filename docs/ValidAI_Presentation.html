Finding Severity Levels

Weighted scoring: Critical×10, High×5, Medium×2

  • Critical — Immediate security or correctness risk. Blocks quality gate.
  • High — Significant issue requiring attention before merge.
  • Medium — Code smell or moderate risk. Should be addressed.
  • Info — Informational. Excluded from scores and gates.

Quality Gate Defaults

Min Score: 70% · Max Critical: 0 · Max High: 3 · Max Medium: 10

Key Capabilities

Real-Time Analysis

Debounced auto-analysis as you type with live issue badge. Web Worker pool for non-blocking execution.

Incremental Diff

Only analyze changed lines. Identical resubmissions return instantly. Comparison view for two runs.
